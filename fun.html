HAVE FUN

Games play an essential role in .scratch !! In order to acknowledge and understand some of the trickiest rules of code languages you will be challenging yourself and improving your knowledge. Practicing is key to accomplish your goals. All you need is a boost driven by game excitements. Let's score!!

Flexbox Defense

Your job is to stop the incoming enemies from getting past your defenses. Unlike other tower defense games, you must position your towers using CSS!

CSS Diner

It's a fun game to learn and practice CSS selectors.

Flexbox Froggy

Help Froggy and friends by writing CSS code! This game introducesw the fundamentals of using CSS flexbox to position elements on a webpage.

Grid Garden

In this game, you must water your carrot crops and poison the weeds by mastering CSS grid. With it you can define columns, rows, and grid template areas.
